### Taro CLI 工具安装
首先，你需要使用 npm 或者 yarn 全局安装 @tarojs/cli，或者直接使用 npx:
```bash
# 使用 npm 安装 CLI
$ npm install -g @tarojs/cli

# OR 使用 yarn 安装 CLI
$ yarn global add @tarojs/cli

# OR 安装了 cnpm，使用 cnpm 安装 CLI
$ cnpm install -g @tarojs/cli
```

### 安装项目依赖
``` bash
cnpm i
```

## 运行 程序

``` bash
npm run dev:weapp
```
